URL Spoofing Trick 2 - Random/Unrelated/Trustworthy Domain, without Brand

This trick uses random/unrelated or trustworthy names or strings as domain name and does not include the brand name of the targeted website in any other part of the URL.
Example:- http://www.szuhsa.fr/login.html, http://www.weather.com/ login.html or https://secure-payment.com to impersonate PayPal. You can see that Paypal is not included in any part of the URL.
