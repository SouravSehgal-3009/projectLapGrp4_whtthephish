URL Spoofing Trick 4 - Random/Unrelated/Trustworthy/IP Domain, with Brand in Path

A Phisher can include the brand name into the path part of a URL in combination with a random/unrelated/trustworthy domain name

Example:-http://onlinepayment.com/www.paypal.com/. This attack can also happen in combination with an IP address URL, e.g. http://5.178.64.164/paypal.
